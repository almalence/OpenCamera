Main controls

Bottom controls

Left control - fast access to your latest image in the gallery/editor. 

Center control - shutter to start/stop capture.

Right control - mode selection.

Top controls

Top menu shows fast controls. Pull it down to see all quick controls:

Scenes, white balance, ISO, flash, focus, exposure metering and more!

Barcode scanner control to enable/disable scanner. When enabled - place barcode/QR code into frame and wait for a second until scanned.

Select "more settings" to get access to all settings - general, viewfinder, capture and processing modes, export.

Self-timer control

Show/hide self-timer and time-lapse control.

Select time interval and other parameters to take photos with self-simer.

Switch to time-lapse tab to configure time-lapse capture. Device will go into sleep mode if interval between photos is big enough.
