More settings

When in settings - you will find different sections to access all application settings:

General

Access to all image sizes in any mode - general image size, multishot and night image size, panorama height, video resolution.

- switching to new camera API and enabling RAW support if available

- volume button and head-set button settings

- image quality settings

- options to take image on single/double click

- color effects

- hardware camera button configuration if available

- external gallery selection

- anti-banding

- camera parameters - this option useful if you contact support

Viewfinder

Settings in this section allow to configure controls on main screen and manage screen behavior.

You can add:

- self timer control

- auto exposure and white balance lock controls

- zoom control

- histogram

- barcode scanner

- level

- info controls to the main screen (scene, white balance, shutter speed, exposure compensation, focus, flash, ISO, battery status, number of images left)

Shooting modes

Settings in this section allow to configure all parameters of capture and processing for all modes.

Saving and sharing

Allow to set saving options - storage location, format, gps tags, timestamp options, image orientation.
